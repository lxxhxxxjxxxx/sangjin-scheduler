## 핵심 비즈니스 로직

### 시간 버는 활동 (배율)
| 활동 | 배율 | 부모 확인 |
|------|------|----------|
| 학원/과외 공부 | 1배 | X |
| 숙제 | 1배 | X |
| 스스로 공부 | 1.5배 | O (엄마) |
| 독서 + 독후감 | 1.5배 | O (엄마) |
| 좋은 일 | 1.5배 | O (엄마) |
| 코딩/AI | 2배 | O (아빠) |
| 앱 완성 | +100시간 | O (아빠) |
| 앱스토어 배포 | +1000시간 | O (아빠) |

### 시간 쓰는 활동
| 활동 | 소비 |
|------|------|
| 게임 | 1:1 |
| 유튜브 | 1:1 |
| 아이템 교환 | 10시간 → 3만원 |

### 벌금
| 사유 | 벌금 |
|------|------|
| 미기록 | -1시간 |
| 잔액 없이 사용 | -2시간 |
| 거짓말 | -10시간 |
